# APLIKASI PEMINJAMAN ALAT

## Deskripsi
Aplikasi untuk mengelola peminjaman alat dengan 3 level pengguna:
1. **Admin** - CRUD User, Alat, Kategori, Laporan
2. **Petugas** - Menyetujui peminjaman, memantau pengembalian
3. **Peminjam** - Melihat alat, mengajukan peminjaman, riwayat

## Fitur Utama
- Login/Logout untuk semua role
- Manajemen user (CRUD)
- Manajemen alat dan kategori
- Peminjaman dan pengembalian alat
- Perhitungan denda otomatis
- Laporan lengkap
- Log aktivitas

## Teknologi
- PHP 7.4+
- PostgreSQL
- Bootstrap 5
- Font Awesome 6
